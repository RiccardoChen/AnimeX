# Animex_front-end

This is the front-end of the Animex project, built with **Angular** [16.2.12](https://v17.angular.io/guide/update-to-version-16).

Key tools and technologies used in this project:

- **TypeScript** [5.1.3](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-1.html)
- **Tailwind** [3.4.17](https://v3.tailwindcss.com/)
- **SweetAlert2** [11.22.0](https://sweetalert2.github.io/#download)
- **PrimeIcons** [v21](https://primeng.org/icons)

## Setup ðŸ”¨

1. After cloning the repository to your local machine, navigate into the project directory.

   ```bash
   cd Animex/Animex_front-end
   ```

2. Install the project dependencies (ensure you have the **Node.js** LTS version installed first).

   ```bash
   npm install
   ```

3. Launch the development server.
   ```bash
   npx ng serve --open
   ```

#### **_NOTE_** : To run the complete project, you must start both the front-end and back-end servers simultaneously.
